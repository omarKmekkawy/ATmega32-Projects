# Atmega32_LCD
This example shows how to use the LCD and print int and float variables.

## LCD Connection

```
LCD -> mcu
RS  : PB0
EN  : PB1
RW  : PB2
D0  : PD0
D1  : PD1
D2  : PD2
D3  : PD3
D4  : PD4
D5  : PD5
D6  : PD6
D7  : PD7
```
